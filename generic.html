Zoom Integration

Consumed the Zoom API POST method to create a meeting. Using ServiceNow I created an application called Zoom meeting and in each record there is a UI action ,form button that opens a new UI page/Modal , allows the user to input details such as start date, duration and title of meeting. After the submit a request is done and a new section in the incident appears where the meeting URL ,start time ,password of the meeting is shown.
