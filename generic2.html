Stock App

Created an application from scratch using an API for stocks where I got the latest close and open prices of selected company stocks. The goal of this application was to take the data using REST operation and create reports so that I can quickly check my favorite updated stock prices on my ServiceNow homepage or dashboard. I created a Scheduled Job where once a day it sends a request to the API and it returns close and open day prices.
